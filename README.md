## Academic Papers

| API | Papers | Key Required | Rate Limit | Description |
|-----|--------|-------------|------------|-------------|
| [OpenAlex](https://docs.openalex.org/) | 250M+ | ❌ No | 100K/day | Open catalog of scholarly works, authors, institutions |
| [arXiv](https://arxiv.org/help/api) | 2M+ | ❌ No | Generous | Preprints in physics, CS, math, biology |
| [Semantic Scholar](https://api.semanticscholar.org/) | 200M+ | ✅ Yes (free) | 100/5min | AI-powered academic search |
| [CORE](https://core.ac.uk/services/api) | 200M+ | ✅ Yes (free) | 10K/day | Open access research papers |
| [Crossref](https://api.crossref.org/) | 130M+ | ❌ No | 50/sec | DOI metadata, citations |
| [Unpaywall](https://unpaywall.org/products/api) | 30M+ | ❌ Email only | Generous | Find free legal PDFs of papers |
| [ORCID](https://info.orcid.org/documentation/api-tutorials/) | 17M+ researchers | ❌ No | Generous | Researcher profiles and publications |
